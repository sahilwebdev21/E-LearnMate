# E-LearnMate - E-learning Platform

## Overview
An end-to-end learning platform with features like:
- Student/Instructor/Admin dashboards
- Course management
- Secure video streaming
- Payments
- Certificates

## Project Structure
- frontend/
- backend/
- database/
- docs/

## Stack
Frontend: Next.js + TypeScript  
Backend: Django + DRF  
Database: PostgreSQL  
Deployment: Vercel + Render  

## Features
👥 User Roles
* Student: Browse/buy courses, track progress, attempt quizzes, get certificates.

* Instructor: Create/upload courses, view student progress, earnings.

* Admin: Manage users, approve content, view analytics & revenue.

🎯 Main Features
* Authentication (JWT)

* Course management (CRUD)

* Video lectures

* Quizzes

* Progress tracking

* Payments (Stripe/Razorpay)

* Certificate generation

* Reviews, leaderboard (optional)
